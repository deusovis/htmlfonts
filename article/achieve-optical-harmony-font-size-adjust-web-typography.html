Achieve Optical Harmony with `font-size-adjust`

Stop matching font sizes by visual appearance alone; instead, ensure consistent x-heights for harmonious inter-font typography. Use `font-size-adjust` to programmatically normalize different font's actual body sizes, creating a unified optical rhythm across your UI.

/* Ensure optical size consistency across diverse web fonts */
body {
  font-family: 'Inter', sans-serif;
  font-size: 1rem;
  /* Example: Inter's x-height ratio might be ~0.52 */
}

h1 {
  font-family: 'Playfair Display', serif; /* A font with a naturally different x-height */
  font-size: 2.5rem;
  font-size-adjust: 0.52; /* Set to the target x-height ratio (e.g., from your body font) */
  /* This scales Playfair Display's font-size so its x-height visually matches 52% of its computed font-size,
     creating better optical balance with other fonts like Inter, even at different declared font-sizes. */
}
