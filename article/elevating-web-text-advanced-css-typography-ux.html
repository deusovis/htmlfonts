Elevating Web Text: Advanced CSS Typography for Better User Experience

For a truly superior user experience, combine `font-display: swap;` for optimal web font loading performance with modern fluid typography using `clamp()`. `font-display: swap;` prevents blank text flashes (FOIT) by immediately rendering fallback fonts, then swapping in your custom web font once loaded. Simultaneously, `clamp()` allows you to define a minimum, preferred, and maximum font size, enabling fluid scaling that adapts beautifully across screen sizes without complex media queries. This dual approach ensures both visual consistency and excellent legibility, significantly enhancing your UI's responsiveness and perceived speed.

/* 1. Optimize Web Font Loading with font-display */
@font-face {
  font-family: 'Inter-Regular';
  src: url('/fonts/Inter-Regular.woff2') format('woff2');
  font-weight: 400;
  font-style: normal;
  font-display: swap; /* Critical for perceived performance and UX */
}

/* 2. Implement Fluid Typography with clamp() for Responsive Sizing */
body {
  font-family: 'Inter-Regular', sans-serif;
  /* clamp(minimum, preferred, maximum) */
  /* Example: Font size fluidly scales between 1rem (16px) and 1.25rem (20px) */
  font-size: clamp(1rem, 2vw + 0.5rem, 1.25rem);
  line-height: 1.6;
  color: #2c3e50;
}

h1 {
  /* Example: Heading size fluidly scales between 2rem (32px) and 3.5rem (56px) */
  font-size: clamp(2rem, 5vw + 1rem, 3.5rem);
  line-height: 1.15;
  margin-bottom: 0.75em;
}
